# TrevalTrip

Mvc teknolojilerini aktif kullanma odaklı, seyahatlar hakkında bir blog sitesi proje. Sitenin yayına hazır hali trevaltrip klasöründe; kaynak kodları ise source codes
bölümündedir. Üstelik sitedeki tüm içeriği düzenlemeye yardımcı olmak için bir admin panel de vardır.
